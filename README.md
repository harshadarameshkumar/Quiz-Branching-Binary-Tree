# Quiz Branching Scenario App Using Binary Tree

## Description
This project demonstrates the use of Binary Trees in C to create an interactive
branching quiz. Each user response determines the next question, forming a unique path.

## Data Structure Used
- Binary Tree
- Pointers
- Dynamic Memory Allocation

## How It Works
- Each node represents a question or result
- Yes/No answers traverse left/right branches
- Leaf nodes store final outcomes
